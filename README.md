# Eco Credit App ♻️

Eco Credit App is a Flutter-based mobile and web application designed to encourage sustainable waste disposal using AI-based waste classification and reward points.

## Features

* User login & registration
* Waste disposal tracking
* AI waste classification integration
* Reward points system
* Leaderboard
* Admin dashboard
* Firebase authentication
* Firestore database
* Flutter Web + Android support

## Technologies Used

* Flutter
* Firebase Auth
* Cloud Firestore
* Python AI model
* OpenCV
* TensorFlow
* Node.js (Firebase functions)

## How to run

```
flutter pub get
flutter run
```

## Project Purpose

This project was developed as part of B.Tech final year project:
Waste Segregation through AI Sensors – Sustainable Practices Implementation in Educational Institutions.
